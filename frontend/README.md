# OptiResume AI - Frontend (React)

A cinematic, high-performance user interface for real-time resume optimization and ATS simulation.

## ✨ Highlights
- **Cinematic UI**: Premium dark-mode design with Framer Motion animations.
- **Bento Dashboard**: Data-dense views for ATS scores, skill gaps, and resume comparisons.
- **Dynamic Loader**: Intelligent multi-stage AI loading sequences.
- **Responsive Web**: Optimized for all devices from mobile to desktop.

## 🛠️ Setup & Installation

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Configuration**:
   Create a `.env` file in this directory:
   ```env
   VITE_API_URL=https://your-backend-url.onrender.com/api/v1
   ```

3. **Development Mode**:
   ```bash
   npm run dev
   ```

4. **Production Build**:
   ```bash
   npm run build
   ```

## 🎨 Design System
- **Framework**: Tailwind CSS
- **Interactions**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Inter / Roboto

## 🚀 Key Features
- **Real-time ATS Meter**: Animated score visualization.
- **Interactive Skill Gaps**: Categorized missing keywords (High/Medium/Low priority).
- **Side-by-Side Comparison**: Visual diff of AI-optimized changes.
- **Dual-Format Downloads**: Seamless handling of PDF and DOCX exports.
